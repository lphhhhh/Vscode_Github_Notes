## 三.字符串的朴素模式匹配算法

 ````c++
Index(SString S,SString T)
{
    int i=1,j=1;      //i指向S上的字符，j指向T上的字符
    while (i<=S.length&&j<=T.length)
    {
        if(S.ch[i]==T.ch[j])
            ++i;  ++j;
    }
    else
    {
        i=i-j+2;
        j=1;
    }
    if(j>T.length)          //j已经将T全部扫描，说明已经匹配完全
        return i-T.length     //此时i指向匹配到的字符串的下一个字母，-length即为首字母
    else
        return          //匹配失败
}
    
 ````

1. 朴素算法中没有任何记忆功能，i和j都将S和T重复扫描
2. 时间复杂度：mn
